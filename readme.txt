Steps to run App 
1) open app in Visual studio open a terminal and run command:  ng serve
2) You should have mongodb install and setup on your pc, if yes the run mongod on cmd.
3) open another terminal in visual studio code and go to ServerSideApp folder using command "cd ServerSideAppp" 
3) In ServerSideApp terminal run command npm start. ( this will start server on port 3500)
4) Open browser and write http://localhost:4200
5) make  user account because without it you can't access todolist.
6) Login 
8) Start use TodoApp
